# Opening_Nursery_For_Mac
 开局托儿所自动化mac版

[windows点这里](https://github.com/SakuraHentai/nursery-bot)

## 准备

需要 **Python3.8+** 

安装 **tesseract**
`brew install tesseract` 

安装 **pyautogui** 
`pip3 install pyautogui`

安装 **cv2**
`pip3 install opencv-python`

安装 **numpy**
`pip3 install numpy`

安装 **pytesseract**
`pip3 install pytesseract`

## 使用
打开开局托儿所，点击开始游戏，然后运行run_me.py

无分辨率限制

可能会识别失败，放弃这局，重新开始即可
